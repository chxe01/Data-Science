## **FIFA ⚽️**

### 1. 프로젝트 개요
구단의 합리적인 영입 예산 계획을 위해 FIFA 선수 능력치 데이터를 활용하여 선수 몸값을 예측하는 회귀 분석 프로젝트

### 2. 사용 데이터
player_stats.csv - Kaggle

### 3. 분석 목적
선수 능력치와 몸값 간의 관계를 파악하여 적정 몸값을 예측하고, 데이터 기반 구단의 선수 영입 결정을 지원하기 위해 분석하였다.

### 4. 사용 기술 스택
Python Numpy Pandas

### 5. 주요 인사이트/결과
몸값에 가장 큰 영향을 준 능력치는 reactions(상황 판단 속도) (0.156)으로, 포지션에 관계없이 모든 선수에게 요구되는 핵심 역량임을 확인하였다,
그 다음으로 shot_power(0.045), short_pass(0.035) 가 선수의 몸값에 영향을 주는 걸 알 수 있었다.

### 6. 실행 방법
1. Jupyter Notebook 설치
2. 라이브러리 설치
   pip install pandas numpy scikit-learn
3. Kaggle에서 player_stats.csv 다운로드
4. fifa.ipynb 실행

### 7. 트러블 슈팅
value 컬럼의 분포를 시각화했을 때 심한 우편향을 확인했고, 선형회귀의 특성상 이를 보정하기 위해 log 변환을 적용했다. 결과적으로 R²가 0.24 → 0.70으로 향상됐다.

gk_positioning 컬럼이 실수로 포함됐을 때 R² 0.70, 제거 후 0.68 — 논리적 일관성을 위해 제거 선택
